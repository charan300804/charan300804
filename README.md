## 🏆 Featured Projects

### 🎮 [Autonomous Hill Climb Racing AI](https://github.com/charan300804/AI-HILL-CLIMB-RACING)
An AI agent that plays the 2D game *Hill Climb Racing* using computer vision (**YOLOv8**) to perceive the screen and reinforcement learning (**DQN**) to make real-time decisions via keyboard simulations.

### 🔍 [HGT Fake Review Detection GNN](https://github.com/charan300804/hgt-fake-review-detection)
An end-to-end fraud detection system modeling e-commerce behaviors as a heterogeneous graph and using a **Heterogeneous Graph Transformer (HGT)** in PyTorch Geometric to detect review rings.

### 🏭 [ComminuTech AI (SIH2)](https://github.com/charan300804/SIH2)
An advanced Next.js web application utilizing **Google Genkit AI** and Firebase to monitor and optimize energy efficiency and predict equipment faults in heavy mining circuits.

### 🚀 [GigFlow Marketplace](https://github.com/charan300804/GigFlow)
A premium full-stack freelance marketplace featuring real-time socket communication, state management using Redux Toolkit, and secure authentication.
